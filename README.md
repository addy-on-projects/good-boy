# good-boy
Good Boy is a Python-based AI voice assistant that listens for the wake word "Good Boy" before accepting commands. Once activated, it can recognize voice input, perform various tasks, and return to standby mode until called again.

Features
🎙️ Wake word detection using "Good Boy"
🗣️ Speech recognition for voice commands
🌐 Open Google instantly
🎵 Play songs on command
😴 Exit or stop listening with the "Sleep" command
⚡ Hands-free interaction with fast command execution
🐍 Built using Python and speech recognition libraries
